## React Page Builder

this project created with vite, React, typescript, DndKit, Redux, Tailwind, json-server, nginx

## Installation

This project is dockerized

### development
run as follows:

```bash
cd './Docker/Build'
docker-compose up --build
```

project is up and running on port 80

### Production
run as follows:

```bash
cd './Docker/Production'
docker-compose up --build
```

project is up and running on port 80
